Project: Efficient Codebase Exploration Skill

Description:
A reusable AI-agent skill that teaches efficient codebase navigation:

* Search-first, read-second workflow
* Parallel tool calls for independent operations
* Targeted file reading with offset/limit
* Context and structure documentation management
* Exploration limits with escalation rules
* Minimal edit principles
* Anti-pattern avoidance
